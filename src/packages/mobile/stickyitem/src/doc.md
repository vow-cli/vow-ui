# stickyItem 吸顶容器子组件

## API

### Props

| 参数 | 说明                               | 类型   | 默认值 |
| ---- | ---------------------------------- | ------ | ------ |
| top  | 吸顶容器吸顶后举例参照物顶部的距离 | Number | 0      |

### Events

| 事件名    | 说明               | 回调参数                      |
| --------- | ------------------ | ----------------------------- |
| load-more | 继续加载的回调函数 | done 函数，用于关闭加载中状态 |
